    Instant Uploads

    You have complete control of your image uploads.

    • Configurable Image host, Photo album, Authentication
    • Signal: wifi, cellular, signal strength, roaming
    • Power: plugged in, battery level
    • Plugins: Tasker/Locale and DashClock

    Plugins

    PixelRelay supports Tasker and DashClock

    Tasker integration allows you to configure tasks that can change the photo album, private key and enable/disable automatic uploads.

    DashClock plugin supports current pending uploads and current upload progress

    Plugin Variables

    PixelRelay supports the following variables in the Album name

    %DATE
		Replaces the variable with the current date
		in `yyyy-MM-dd` format.
		
    %DATE(format)
		Replaces the variable with the current date
		in a customized format.
		Example: `dd-MM-yyyy` would become `01-08-2013`
     

    QR Code

    QR codes can be used to change the host, album and private key. This is useful for events where multiple people may want to quickly and easily use the same settings.

    		QR Format:
		PixelRelay://scan?host=HOSTURL&album=ALBUMNAME&privatekey=PRIVATEKEY

    API

    The PixelRelay Android app provides the following params when uploading

    String version
		PixelRelay Android app version information.
    File uploaded_file
		The binary file being uploaded.
    String user_email 
		The google account associated with the upload.
    String user_private_key
		The private code to allow upload access.
    String file_host
		The image host entered in the android app.
		Provided for server debug purposes.
    String file_album
		The album name the user would like
		the image(s) to be added to.
    String file_name
		The file's name from Android.
    String file_mime
		The file's mime-type from Android.
