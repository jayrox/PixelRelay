    Instant Uploads

    You have complete control of your image uploads.

    • Configurable Image host, Photo album, Authentication
    • Signal: wifi, cellular, signal strength, roaming
    • Power Conditions: plugged in, battery levels
    • Plugins: Tasker/Locale and DashClock

    Plugins

    PixelRelay supports Tasker and DashClock

    Tasker integration allows you to configure tasks that can change the photo album and enable/disable automatic uploads.

    DashClock plugin supports current pending uploads and current upload progress

    Installation

    The example PixelRelay upload code is written in PHP to allow the most possible server configurations.

    API

    The PixelRelay Android app provides the following params when uploading

    uploaded_file
		The binary file being uploaded.
    user_email 
		The google account associated with the upload.
    user_private_key
		The private code to allow upload access.
    file_host
		The image host entered in the android app.
		Provided for server debug purposes.
    file_album
		The album name the user would like
		the image(s) to be added to.
    file_name
		The file's name from Android.
    file_mime
		The file's mime-type from Android.
